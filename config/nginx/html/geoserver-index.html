Выбор инстанции GeoServer

Выберите инстанцию GeoServer для доступа:

Vector GeoServer

Инстанция GeoServer для работы с векторными данными.

Доступные функции:

  • WMS/WFS сервисы
  • Векторные слои
  • Стилизация
Открыть Vector GeoServer

ECW GeoServer

Инстанция GeoServer для работы с растровыми данными ECW.

Доступные функции:

  • WMS/WCS сервисы
  • Растровые слои ECW
  • Оптимизированный рендеринг
Открыть ECW GeoServer
