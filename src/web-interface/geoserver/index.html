Выбор GeoServer

Выберите инстанцию GeoServer, с которой хотите работать:

GeoServer Vector

Оптимизирован для работы с векторными данными (Shapefile, PostGIS, GeoJSON)

Данная инстанция предназначена для управления векторными данными и публикации WFS сервисов.

Перейти к Vector GeoServer

GeoServer ECW

Специализирован для работы с растровыми данными (ECW, GeoTIFF)

Данная инстанция предназначена для работы с растровыми данными большого объема и публикации WMS сервисов.

Перейти к ECW GeoServer
